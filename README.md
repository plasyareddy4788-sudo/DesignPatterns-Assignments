1. Creational Patterns
Singleton Pattern (SingletonPattern.java)

Use Case: Ensures a logging utility class has only one instance throughout the application lifecycle.

Factory Method Pattern (FactoryMethodPattern.java)

Use Case: Dynamically creates different types of documents (Word, PDF, Excel) using a unified factory interface.

Builder Pattern (BuilderPattern.java)

Use Case: Manages the step-by-step construction of complex objects like a Computer with optional parts.

2. Structural Patterns
Adapter Pattern (AdapterPattern.java)

Use Case: Integrates different third-party payment gateways (PayPal, Stripe) into a uniform payment processing system.

Decorator Pattern (DecoratorPattern.java)

Use Case: Dynamically wraps notification methods to send alerts across multiple channels (Email, SMS, Slack) simultaneously.

Proxy Pattern (ProxyPattern.java)

Use Case: Optimizes an image viewer application by adding lazy loading and memory caching for remote images.

3. Behavioral Patterns
Observer Pattern (ObserverPattern.java)

Use Case: A stock market application that automatically alerts registered mobile and web clients when stock prices change.

Strategy Pattern (StrategyPattern.java)

Use Case: Allows a system to switch payment execution methods (Credit Card or PayPal) dynamically at runtime.

Command Pattern (CommandPattern.java)

Use Case: A home automation system remote control that decouples requests from execution using turn-on and turn-off commands.

4. Architectural Patterns & Principles
MVC Pattern (MVCPattern.java)

Use Case: Separates a student record management application into Model (Data), View (UI), and Controller (Logic).

Dependency Injection (DependencyInjectionPattern.java)

Use Case: Uses constructor injection to decouple a Customer Service layer from its database repository layer.
